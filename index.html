Hello!

My name is Nathan, and I am a self-taught Web Developer! I started teaching myself in September 2015 while working full-time and planning a wedding.

My main focus thus far has been Javascript and related technologies. I feel comfortable explaining how Javascript works "under the hood". Being able to comprehend source code has made learning JS based technologies much easier.

Working with me is quite simple. I have a personal creed that I follow, for work, and home life. This applies when I work in a team, receive solo tasks, and even when I am doing personal projects. There are 3 rules.

  1. Communicate
  2. Apply
  3. Refine
  4. Repeat

Like my little joke? Moving on!

Communicate

To effectively communicate tasks and expectations and convey useful abilities and skills. This creates clear, tangible goals.

Apply

To use my abilites to achieve those goals. This includes my ability to research and adapt.

Refine

To reflect on what the task has taught me, and attempt to become a more efficient and useful developer. This applies to integrating myself into a team, as well.

Repeat

As I learn I must communicate my new skills, apply those skills if new tasks are assigned, and refine my abilites once again.
